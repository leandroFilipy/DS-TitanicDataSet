# 🚢 Titanic Dataset Analysis

> Uma análise completa do conjunto de dados do Titanic com Python

---

## 📋 Sobre o Projeto

Este repositório contém uma análise exploratória e modelagem preditiva do famoso conjunto de dados do Titanic. O objetivo é prever a taxa de sobrevivência dos passageiros baseado em características demográficas e sociais.

---

## 📊 Dataset

O dataset inclui informações sobre:

- 👥 **Passageiros**: 891 registros de treinamento
- 🎫 **Características**:
  - `PassengerId` - ID único do passageiro
  - `Survived` - Se o passageiro sobreviveu (0 = Não, 1 = Sim)
  - `Pclass` - Classe da cabine (1ª, 2ª ou 3ª)
  - `Name` - Nome do passageiro
  - `Sex` - Gênero (male/female)
  - `Age` - Idade
  - `SibSp` - Número de cônjuges/irmãos a bordo
  - `Parch` - Número de pais/filhos a bordo
  - `Ticket` - Número da passagem
  - `Fare` - Valor da passagem
  - `Cabin` - Número da cabine
  - `Embarked` - Porto de embarque (C, Q, S)
